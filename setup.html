Dán code vào Apps Script

Trong Sheet: bấm menu Extensions → Apps Script → xóa code cũ → dán code bên dưới → bấm 💾 Save.

function doPost(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); if (sheet.getLastRow() === 0) { sheet.appendRow(['Thời gian', 'Tên', 'Email', 'Tin nhắn']); } const d = JSON.parse(e.postData.contents); sheet.appendRow([new Date().toLocaleString('vi-VN'), d.name||'', d.email||'', d.message||'']); return ContentService.createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); } function doGet() { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); const rows = sheet.getDataRange().getValues(); return ContentService.createTextOutput(JSON.stringify({ok:true, rows:rows})) .setMimeType(ContentService.MimeType.JSON); }
💡 Sau khi Save: bấm Deploy → New deployment → Web app → Execute as: Me → Who has access: Anyone → bấm Deploy → copy URL.
